RunWalk Coach Privacy Policy

Last updated: June 22, 2026

Data Stored

RunWalk Coach stores workout sessions, training settings, recovery-code metadata, and anonymous session records. Workout reports can include manually entered heart-rate values, perceived difficulty, pain, breathing level, and notes.

Local Storage

The app stores pending workout sessions in your browser's IndexedDB so sessions can be saved offline and synced later.

Account Recovery

Recovery codes are shown to you in the app. The server stores only a hashed version of the code. Anyone with the recovery code can restore access to the same anonymous profile.

Deletion

You can delete server and local progress from Settings. Deletion removes the anonymous profile, workout sessions, template overrides, sessions, and recovery code metadata associated with the current browser session.
